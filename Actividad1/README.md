# Sistema de Inventario en PHP (POO)

## Descripcion
Este proyecto implementa un sistema de gestión de inventario utilizando **Programación Orientada a Objetos (POO)** en PHP.  
Permite **agregar productos**, **listar productos** y **actualizar el stock** de los mismos dentro de una categoria.

## Caracteristicas
- Clase `Producto` con atributos privados y métodos de acceso par entrelazarlos con index y categoria.
- Clase `Categoria` que administra una colección de productos del sistema.
- Encapsulamiento y validaciones en constructores.
- Manejo de errores mediante `try-catch`.
- Interfaz visual sencilla en `index.php` con formularios HTML.
- Estilos style.css para la parte visual del acabado.

##  Requisitos
- PHP 7.4 o superior.
- Navegador web de su preferencia.
- Servidor local (XAMPP, **Laragon**, Wamp, etc.).

## Ejecucion
1. Clonar o descargar el repositorio.
2. Abrir la carpeta en un servidor local.
3. Acceder a `http://localhost/Actividad1/index.php`.

##  Ejemplo de uso
1. Agregar un producto con nombre, precio y stock.
2. Verlo reflejado en la tabla.
3. Actualizar el stock desde el formulario.
4. Si intentas actualizar un producto inexistente, se mostrará un mensaje de error.
